# Training History

## Overview
코치가 훈련 완료 후 수동으로 기록을 남기는 기능.
"오늘 뭘 했는지" 히스토리가 쌓여서 나중에
코치 프로필과 훈련 패턴 파악에 활용된다.

## User Flow
1. 훈련 플래너에서 해당 날짜 훈련 열기
2. "훈련 완료" 버튼 탭
3. 실제 진행한 드릴 확인/수정 (계획과 달랐을 경우)
4. 참석 인원 입력
5. 코치 메모 작성 (선택)
6. 저장

## Data Model
| 필드 | 타입 | 필수 여부 |
|------|------|-----------|
| 날짜 | date | ✅ 필수 |
| 진행한 드릴 목록 | drill[] | ✅ 필수 |
| 참석 인원 | number | ✅ 필수 |
| 코치 메모 | textarea | ⬜ 선택 |
| 완료 여부 | boolean | ✅ 자동 |

## UX 요구사항
- 플래너에서 "완료" 한 번으로 기록 생성 가능
- 계획과 실제가 달랐을 때 드릴 수정이 쉬워야 함
- 히스토리는 캘린더 또는 리스트 뷰로 열람 가능
