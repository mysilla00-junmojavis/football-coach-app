# Training Planner

## Overview
코치가 드릴을 조합해서 하나의 훈련 세션을
계획하는 기능. 빠르게 짜고 바로 현장에서
꺼내볼 수 있어야 한다.

## User Flow
1. 날짜 선택
2. 내 드릴 라이브러리에서 드릴 검색 & 추가
3. 드릴 순서 드래그로 정렬
4. 총 소요시간 자동 계산 & 확인
5. 저장 → 훈련 기록으로 연결

## UX 요구사항
- 드릴 추가가 2탭 이내로 가능해야 한다
- 총 소요시간이 실시간으로 보여야 한다
- 저장된 플랜을 나중에 그대로 복제해서
  재사용할 수 있어야 한다

## Data Model
| 필드 | 타입 | 설명 |
|------|------|------|
| 날짜 | date | 훈련 예정일 |
| 드릴 목록 | drill[] | 순서 포함 |
| 총 소요시간 | number (분) | 자동 계산 |
| 메모 | textarea | 오늘 훈련 특이사항 |
