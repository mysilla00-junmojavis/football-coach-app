# Drill Creation

## Overview
코치가 드릴을 직접 만들고 저장하는 기능.
빠르게 입력하고, 나중에 훈련 플래너에서
바로 꺼내 쓸 수 있어야 한다.

## Fields

| 필드 | 타입 | 필수 여부 | 설명 |
|------|------|-----------|------|
| 이름 | text | ✅ 필수 | 드릴 제목 |
| 목적 | multi-select | ✅ 필수 | 패스/슈팅/수비/드리블 등 |
| 소요시간 | number (분) | ✅ 필수 | 예: 15분 |
| 참여 인원 | number or range | ✅ 필수 | 예: 6~10명 |
| 난이도 | select | ✅ 필수 | 초급/중급/고급 |
| 설명 텍스트 | textarea | ✅ 필수 | 드릴 진행 방법 |
| 필요 장비/콘원 | multi-select | ⬜ 선택 | 콘/마커/골대/조끼 등 |

## 다이어그램

- v2.0: 이미지 업로드 또는 드로잉 캔버스 고려

## 목적 카테고리 (태그)
패스 | 슈팅 | 수비 | 드리블 | 피지컬 | 전술 | 골키퍼 | 웜업

## UX 요구사항
- 입력 항목이 한 화면에 다 보여야 한다 (스크롤 최소화)
- 필수 항목만 채워도 저장 가능해야 한다
- 저장 후 바로 훈련 플래너에서 사용 가능해야 한다
