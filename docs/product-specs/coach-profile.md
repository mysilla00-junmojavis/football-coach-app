# Coach Profile

## Overview
코치의 활동 현황을 한눈에 볼 수 있는 페이지.
내가 만든 드릴과 훈련 패턴을 파악할 수 있다.

## Sections

### 1. 기본 정보
| 필드 | 타입 | 필수 여부 |
|------|------|-----------|
| 이름 | text | ✅ 필수 |
| 프로필 사진 | image | ⬜ 선택 |

### 2. 내 드릴 목록
- 내가 만든 드릴 전체 리스트
- 최신순 정렬 (기본값)
- 탭에서 바로 드릴 상세 열람 가능

### 3. 드릴 카테고리 통계
- 내가 만든/사용한 드릴을 카테고리별로 집계
- 예: 패스 40% | 슈팅 30% | 수비 20% | 기타 10%
- 시각적 차트로 표시 (바 또는 도넛 차트)

## UX 요구사항
- 프로필은 설정(Settings)에서 편집
- 통계는 자동 계산 (코치가 직접 입력 불필요)
