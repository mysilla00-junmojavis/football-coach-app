# Plans

## MVP 목표
축구 코치가 드릴을 만들고, 훈련을 계획하고,
기록을 남길 수 있는 웹앱을 만든다.

---

## Phase 1 — 기반 세팅 (Foundation)
> 에이전트가 코드를 짜기 전에 반드시 완료

- [ ] Next.js 14 프로젝트 초기화
- [ ] Tailwind CSS + shadcn/ui 설치 및 설정
- [ ] Supabase 프로젝트 생성 및 연결
- [ ] Pretendard 폰트 적용
- [ ] 환경변수 설정 (.env.local)
- [ ] 기본 레이아웃 + 하단 탭 네비게이션 구현

---

## Phase 2 — 인증 (Auth)
> 코치가 로그인해야 드릴을 만들 수 있음

- [ ] Supabase Auth 설정 (이메일 로그인)
- [ ] 로그인 / 회원가입 페이지
- [ ] 로그인 상태에 따른 라우트 보호
- [ ] 코치 프로필 초기 생성 (가입 시 자동)

---

## Phase 3 — 드릴 생성 (Core)
> 앱의 핵심 기능

- [ ] drills 테이블 Supabase 생성
- [ ] 드릴 생성 폼 구현
  - 이름, 목적, 소요시간, 참여인원,
    난이도, 설명, 필요장비
- [ ] 드릴 목록 페이지 (내 드릴)
- [ ] 드릴 상세 페이지
- [ ] 드릴 수정 / 삭제

---

## Phase 4 — 훈련 플래너 (Planner)
> 드릴을 조합해서 오늘 훈련 계획 작성

- [ ] training_plans 테이블 Supabase 생성
- [ ] 훈련 플랜 생성 페이지
  - 날짜 선택
  - 드릴 검색 & 추가
  - 순서 정렬
  - 총 소요시간 자동 계산
- [ ] 플랜 목록 페이지
- [ ] 플랜 상세 페이지
- [ ] 플랜 복제 기능

---

## Phase 5 — 훈련 기록 (History)
> 훈련 완료 후 기록 남기기

- [ ] training_records 테이블 Supabase 생성
- [ ] 플랜에서 "훈련 완료" 버튼
- [ ] 기록 수정 (드릴 변경, 참석 인원, 메모)
- [ ] 훈련 기록 히스토리 리스트
- [ ] 기록 상세 페이지

---

## Phase 6 — 코치 프로필 (Profile)
> 내 활동 현황 한눈에 보기

- [ ] 프로필 페이지 구현
  - 기본 정보 (이름, 사진)
  - 내가 만든 드릴 목록
  - 드릴 카테고리 통계 차트
- [ ] 프로필 편집 기능

---

## Phase 7 — 마무리 & 배포 (Deploy)

- [ ] 전체 플로우 QA
- [ ] 로딩/에러 상태 처리 점검
- [ ] Vercel 배포
- [ ] Supabase RLS(보안 정책) 설정

---

## Post-MVP (v2.0 후보)
- 드릴 커뮤니티 공유
- 팀 관리 (선수 명단, 출석)
- 드릴 이미지 업로드
- AI 훈련 추천
- 훈련 통계 대시보드
