# Product Sense

## Why This Exists
유소년 축구 코치들은 매 훈련마다 드릴을 새로 구성하는 데
불필요한 시간을 낭비한다. 좋은 드릴은 이미 머릿속에 있지만,
그걸 저장하고 재사용하고 다른 코치와 나누는 체계가 없다.

## Core Problem
"오늘 훈련 뭐 하지?" — 코치가 매번 처음부터 고민한다.

## Our Solution
코치가 드릴을 직접 만들어 저장하고, 커뮤니티와 공유하며,
다른 코치의 드릴을 가져다 바로 쓸 수 있는 플랫폼.

## Target User
- **Primary:** 유소년 팀 코치 (아마추어 ~ 세미프로)
- **Secondary:** 없음 (코치 전용)

## Core Value Proposition
"축구 코치를 위한 드릴 라이브러리 & 훈련 플래너 커뮤니티"

## Principles
1. 30초 안에 오늘 훈련을 짤 수 있어야 한다
2. 코치가 IT에 익숙하지 않아도 쓸 수 있어야 한다
3. 한 번 만든 드릴은 영원히 재사용 가능해야 한다
4. 좋은 드릴은 커뮤니티를 통해 자연스럽게 퍼진다

## Success Metrics
- 코치 1인당 주 3회 이상 앱 방문
- 드릴 라이브러리에 1,000개 이상 누적
- 훈련 계획 작성 시간 5분 이하
- 드릴 공유율 (내가 만든 드릴을 공개로 올리는 비율) 30% 이상
